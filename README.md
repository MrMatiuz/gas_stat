# Учет расходов на заправку

Веб-приложение для отслеживания расходов на топливо для различных машин.

## Возможности

- ✅ Запись заправок для разных машин
- ✅ Указание даты, количества литров, цены и пробега
- ✅ Два режима ввода цены: общая сумма или цена за литр
- ✅ Таблица расходов с группировкой по машинам
- ✅ Статистика по всем машинам и отдельно по каждой
- ✅ Расчет расхода топлива (л/100км)
- ✅ Данные сохраняются в браузере (localStorage)

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение в режиме разработки:
```bash
npm run dev
```

3. Откройте браузер и перейдите по адресу, который покажет Vite (обычно http://localhost:5173)

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist`.

## Использование

1. **Добавление заправки:**
   - Перейдите на вкладку "Таблица расходов"
   - Заполните форму: укажите машину, дату, пробег, количество литров и цену
   - Выберите режим ввода: общая сумма или цена за литр
   - Нажмите "Добавить заправку"

2. **Просмотр статистики:**
   - Перейдите на вкладку "Статистика"
   - Увидите общую статистику и статистику по каждой машине

3. **Удаление записей:**
   - В таблице расходов нажмите кнопку "Удалить" рядом с нужной записью

## Технологии

- React 18
- Vite
- LocalStorage для хранения данных
